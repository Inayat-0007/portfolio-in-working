## 🔮 What makes this different?
This isn't your average static site. This portfolio is engineered from the ground up prioritizing **"Zero-Jank"** GPU-accelerated frame rates, brutalist-glass aesthetics, and zero layout shifting. 

Built completely responsive. Rendered on the Edge.

### 🚀 **Key Capabilities:**
- **🎭 Dual Realities:** A completely adaptive context-aware Theme Engine. Flawlessly transitions between a blindingly clean **Light Editorial Mode** and a cyberpunk-inspired **Digital Noir Dark Mode**. 
- **🖥️ 100/100 Lighthouse:** Aggressively tree-shaken and strictly optimized. All animations are hardware-accelerated (`translate3d`), images are optimized across the global Edge via native Next.js image loading, and unnecessary client JS is stripped.
- **✨ Micro-Interactions:** Custom-built Magnetic UI components, dynamic gradient trailing cursors, and fluid spring animations mapped perfectly to user scroll velocity. 
- **🛡️ Military Grade CSP:** Completely stripped of XSS vulnerabilities. Strict `Content-Security-Policy`, `X-Frame-Options`, and `HSTS` headers rigidly enforced natively at the proxy level.
